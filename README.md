# Wynand Janse van Rensburg - CV Website

Personal portfolio and CV website built with HTML, JavaScript, and Tailwind CSS.

## Features

- Responsive design with dark mode toggle and lamp animation
- Dynamic weather effects based on location
- Live Raspberry Pi server stats (CPU, RAM, disk, temperature)
- Interactive project showcase
- Developer console toggle

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Tailwind CSS
- Typed.js

## Hosting

Self-hosted on a Raspberry Pi.
